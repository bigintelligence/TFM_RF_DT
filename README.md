# Requerimientos

- Python 3.10
- pip 23.2.1

# Instalacion y ejecucion
- Crear un entorno virtual
```
python3 -m venv myenv
```
- Instalar *requirements.txt*
```
pip install -r requirements.txt
```
- Ejecutar el script
```
jupyter lab
```
